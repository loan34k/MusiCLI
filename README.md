# MusiCLI

**MusiCLI** is a simple Python command-line interface (CLI) to listen to music directly from YouTube.

---

## Installation

1. Simply run this command (require git) :
```
git clone https://github.com/loan34k/MusiCLI.git && cd MusiCLI && pip install .
```

> For development, you can install it in editable mode :
>
> ```
> git clone https://github.com/loan34k/MusiCLI.git && cd MusiCLI && pip install -e .
> ```

2. Make sure your Python Scripts folder is in your system `PATH` to use the `musicli` command.

---

## System Requirements

* The **`ffplay`** command must be installed on your system.
* To install it, install [**FFmpeg**](https://ffmpeg.org), ([follow the official instructions](https://ffmpeg.org/download.html) or [build it yourself](https://git.ffmpeg.org/ffmpeg.git))

---

## Usage

After installation, simply run :

```
musicli
```
**And enjoy !**
